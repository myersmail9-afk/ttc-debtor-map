# TTC Debtor Map — Brand Integration Guide

This file tells **Claude Code** (or any future editor) exactly where to drop in
Total Tree Care's brand assets so the map is on-brand without touching map logic.

## TL;DR — to re-theme the whole map, edit ONE block

Open `index.html` and find:

```css
/* ============================ BRAND ============================ */
:root {
  --brand-primary:    #2d5a3d;
  --brand-accent:     #6fa66c;
  ...
}
/* ============================ /BRAND ============================ */
```

Change those values. Done. Every header, button, popup, and accent on the page
routes through these variables.

## Variable reference

| Variable               | What it controls                                   | Default      |
| ---------------------- | -------------------------------------------------- | ------------ |
| `--brand-primary`      | Header bar, primary buttons ("Get directions")     | `#2d5a3d`    |
| `--brand-accent`       | Hover states, focus rings on filter chips/search   | `#6fa66c`    |
| `--brand-on-primary`   | Text color shown ON primary backgrounds (usually white) | `#ffffff`    |
| `--brand-text`         | Body text                                          | `#1a1a1a`    |
| `--brand-muted`        | Secondary text (addresses, "X accounts" count)     | `#6b6b6b`    |
| `--brand-bg`           | Page background outside the map                    | `#f5f5f0`    |
| `--brand-surface`      | Filter bar, popup card, search box                 | `#ffffff`    |
| `--brand-border`       | Hairlines around chips, inputs                     | `#e3e3dc`    |
| `--logo-url`           | URL to the logo file (see "Logo" section below)    | `url('logo.svg')` |
| `--logo-height`        | Logo height in header                              | `32px`       |
| `--brand-font`         | Font stack for entire UI                           | system sans  |

## Priority colors — **do not edit unless intentional**

`--pri-red / --pri-orange / --pri-yellow / --pri-green` control the
amount-owed color coding (>$3k / $1.5k–3k / $500–1.5k / <$500). These are
calibrated for color-blind legibility (high contrast against white pins) and
should generally stay as-is. If the brand insists on different priority
swatches, edit them in the same `:root` block — the legend chips and popup
amounts pick them up automatically.

## Logo

1. Drop the logo file (SVG preferred, PNG ok) into the same folder as
   `index.html`. Name it `logo.svg` or `logo.png`.
2. If you renamed it, update `--logo-url` in the `:root` block.
3. **Sizing**: the logo gets rendered into a `<div id="brand-logo">` block.
   Adjust `--logo-height` if needed. The image is `background-size: contain`,
   so it will scale proportionally — give it some horizontal room (the parent
   uses flex with `min-width: 32px`).
4. The fallback text mark (`TTC` in the corner) auto-hides as soon as a
   background image successfully loads. If the logo image is white/light,
   keep the dark `--brand-primary` header behind it; if dark, invert the
   header by swapping `--brand-primary` for a light tone and
   `--brand-on-primary` for dark.

## What NOT to edit

- Anything outside the `BRAND` block in `index.html`.
- `data.csv` — that's the live data source, regenerated by the refresh script.
- Marker geometry, popup HTML, filter logic, fetch URL.

## Adding a favicon

Drop `favicon.ico` next to `index.html` and add this line to `<head>`:

```html
<link rel="icon" href="favicon.ico" />
```

## Adding Open Graph / social preview

If this page ever gets shared (it shouldn't — it has debtor info), add inside
`<head>`:

```html
<meta property="og:title" content="TTC Debtor Map" />
<meta property="og:description" content="Internal — Total Tree Care" />
<meta name="robots" content="noindex, nofollow" />
```

The `noindex, nofollow` is the important one — keep search engines out.

## Privacy posture

This page contains client names + amounts owed + addresses. Treat the
hosting URL as **internal only**. Recommended:

- Use a private GitHub repo with a long, unguessable Pages URL (`projectname-<random>.github.io`), OR
- Host on a password-protected page (Cloudflare Pages Access, Netlify protected sites).

Joseph's deploy currently relies on URL obscurity — flag if the link gets
posted anywhere it shouldn't.
